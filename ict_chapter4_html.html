02

HTML ট্যাগ সমূহ

<tr> ... </tr>

Table Row — টেবিলের একটি সারি/Row তৈরি করে।

<td> ... </td>

Table Data — টেবিলের একটি সাধারণ ঘর/Cell।

<th> ... </th>

Table Highlight — টেবিলের হেডিং সেল। Bold + Centered হয়।

<P> ... </P>

Paragraph — HTML বড় আকারে না থাকলে ব্যবহার হয়।

<u> ... </u>

Underline — লেখার নিচে দাগ দেয়। যেমন: এভাবে

<i> ... </i>

Italic — লেখা বাঁকা হয়। যেমন: এভাবে

<b> ... </b>

Bold — লেখা মোটা/গাঢ় হয়। যেমন: এভাবে

<br>

Line Break — লাইন শেষ করে নিচের লাইনে যায়। Closing tag নেই।

<hr>

Horizontal Rule — Horizontal line আঁকে। Closing tag নেই।

<img>

Image — ছবি যুক্ত করতে। src attribute দিতে হয়।

<a href> ... </a>

Hyperlink — লিংক তৈরি করতে।

<del> ... </del>

Delete — লেখার মাঝখান দিয়ে কাটা দেয়। যেমন: এভাবে

<sup> ... </sup>

Superscript — লেখার উপরে ছোট লেখা। যেমন: x2

<sub> ... </sub>

Subscript — লেখার নিচে ছোট লেখা। যেমন: H2O

colspan

Column Span — একাধিক Column মিলিয়ে একটি Column বানায়।

rowspan

Row Span — একাধিক Row মিলিয়ে একটি Row বানায়। Row/সারির একত্রীকরণ।

📋
<ol type=" "><li></ol>
Ordered List — সংখ্যা বা অক্ষর (১,২,৩ বা A,B,C) দিয়ে বিন্যস্ত তালিকা।
•
<ul type=" "><li></ul>

⚡ Empty Tags (Closing tag নেই)

<br> এবং <hr> — এই দুটো ট্যাগে closing tag থাকে না কারণ এদের কোনো element নেই। এদের Empty Tag বলা হয়।

04

গুরুত্বপূর্ণ সংজ্ঞা

HTML কী?
HyperText Markup Language — ওয়েব ব্রাউজারে প্রদর্শনের জন্য ডিজাইন করা নথিগুলোর আদর্শ মার্কআপ ভাষা।
URL কী?
Uniform Resource Locator — ইন্টারনেটের মাধ্যমে কোনো বিশেষ ওয়েবসাইটে পৌঁছানোর জন্য ব্যবহৃত ওয়েব ঠিকানা।
IP Address কী?
Internet Protocol Address — Website-এর একক ঠিকানা। প্রতিটি ওয়েবসাইটের আলাদা IP থাকে।
Domain কী?
একটি ওয়েবসাইটের নাম বা পরিচয়পত্র।
Tag কী?
একটি বিশেষ Syntax। যা একটি ওয়েবপেজের সামগ্রীর সাথে সম্পর্কিত তথ্য প্রদান করে।
Hosting কী?
ইন্টারনেটে ওয়েবের ফাইলগুলো কোনো সার্ভারে রাখাকে Hosting বলে।
Website কী?
কোনো ওয়েব সার্ভারে রাখা ওয়েব পৃষ্ঠা, ছবি, অডিও, ভিডিও ও অন্যান্য ডিজিটাল তথ্যের সমষ্টি যা ইন্টারনেটে Access করা যায়।
Browser কী?
World Wide Web রিসোর্স থেকে তথ্য খুঁজে বের করা, উপস্থাপন করা এবং সংরক্ষণ করার Application Software।
Search Engine কী?
এমন একটি Software System যা ব্যবহারকারীর প্রশ্নের উত্তরে ওয়েব পৃষ্ঠার Hyperlink সরবরাহ করে।
Web Portal কী?
যেসব link search bar-এর নিচে সদৃশ্যমান থাকে তাকে Web Portal বলে।
Hyperlink কী?
ওয়েবের একটি রিসোর্সে অবস্থিত কোনো ঠিকানা বা Reference।
HTTP কী?
Hyper Text Transfer Protocol — ওয়েব সার্ভার ও ওয়েব Client-এর মধ্যে ডেটা আদান-প্রদান করার Protocol।
Attribute কী?
HTML-এর এলিমেন্টগুলোতে অতিরিক্ত তথ্য প্রেরণ করার জন্য যে পদ্ধতি ব্যবহার করা হয়।
Container Tag কী?
যে সমস্ত ট্যাগে Opening tag এবং Closing tag উভয়ই থাকে তাকে Container বা ধারক ট্যাগ বলে।
WWW কী?
World Wide Web — পৃথিবীর বিভিন্ন দেশের সার্ভারে রাখা পরস্পর সংযোগযোগ্য ওয়েব পেজের সমষ্টি।
Heading কী?
<h1> থেকে <h6> পর্যন্ত ট্যাগের বর্ণনা। <h1> = সর্বোচ্চ, <h6> = সর্বনিম্ন heading।

Static vs Dynamic Website

Static Website Dynamic Website
যেসব ওয়েব পেজের ডেটার মান ওয়েব পেজ চালু করার সময় পরিবর্তন করা যায় না। যেসব ওয়েব পেজের ডেটার মান পেজ চালু করার পর পরিবর্তন করা যায়।
তৈরিতে শুধু HTML ব্যবহার হয়। তৈরিতে PHP, ASP, CSS, XML, JavaScript ব্যবহার হয়।
সাধারণত কোনো Database ব্যবহার হয় না। Database ব্যবহার হয়।
ওয়েবসাইটের Theme ও Content নির্দিষ্ট থাকে। Design ও Content Run Time-এ পরিবর্তিত হয়।

06

গুরুত্বপূর্ণ প্রশ্নোত্তর

HTML কেন ব্যবহার করা হয়? অনুধাবন

HTML ব্যবহার করা হয় কারণ — ১. HTML দ্বারা তৈরিকৃত পেজ সহজে Load হয়। ২. প্রতিটি Browser HTML সমর্থন করে। ৩. শেখা ও ব্যবহার করার পদ্ধতি সহজ। ৪. পেজের সাইজ কম হওয়ায় খরচ কম পড়ে। ৫. User-friendly — Programming সম্পর্কে ধারণা নেই এমন ব্যক্তিও সহজে শিখতে পারে।

Hosting কেন প্রয়োজন? অনুধাবন

কোনো সাইটকে নির্দিষ্ট কোনো সার্ভারে স্থাপন এবং উক্ত ওয়েব সাইটটির যাবতীয় কার্যক্রম পরিচালনার জন্য প্রয়োজনীয় অবকাঠামোগত সুবিধা প্রদান করা হয় ওয়েব Hosting-এর মাধ্যমে। তাই Hosting ওয়েব সাইট Publishing-এর জন্য প্রয়োজনীয়।

ওয়েব পেজ ও Browser-এর সম্পর্ক ব্যাখ্যা করো। অনুধাবন

ওয়েব পেজ হলো এক ধরনের ওয়েব Document যা World Wide Web ও Internet Browser ব্যবহারের জন্য উপযুক্ত। Browser-এর মাধ্যমে ইন্টারনেটে ওয়েব পেজ প্রদর্শনের কাজ করে। পৃথিবীর বিভিন্ন দেশের সার্ভারে রাখা ওয়েব পেজ Browser-এর মাধ্যমে প্রদর্শিত হয়। তাই ওয়েব পেজ ও Browser একে অপরের সাথে সম্পর্কিত।

<br> ও <hr> ট্যাগে closing tag লিখতে হয় না কেন? অনুধাবন

<br> ও <hr> ট্যাগ দুটির Closing tag লিখতে হয় না। কারণ — এসব ট্যাগের Closing tag থাকে না এবং তাদের Element-ও থাকে না। <br> ও <hr> উৎস দুটিকে Empty Tag-ও বলা হয়।

HTML-এ ট্যাগ গুরুত্বপূর্ণ কেন? অনুধাবন

HTML-এ ট্যাগ গুরুত্বপূর্ণ। কারণ — ওয়েব পেজে অনেক ট্যাগ তৈরি করতে HTML ব্যবহৃত হয়। এটি একটি ট্যাগভিত্তিক Language। ওয়েব Design নতুন ব্যবহারকারীদের কাছে খুবই গুরুত্বপূর্ণ। HTML সহজে ব্যবহার করা যায়, পেজ সহজে Load হয়, পেজের সাইজ কম হওয়ায় খরচ কম পড়ে এবং প্রতিটি Browser HTML সমর্থন করে।

CSS কী কাজে ব্যবহার হয়? অনুধাবন

CSS (Cascading Style Sheets) হল একটি Tool যা একটি ওয়েবপেজের বিষয়বস্তুকে তার Visual Design থেকে আলাদা করতে ব্যবহৃত হয়। অর্থাৎ HTML দিয়ে কাঠামো তৈরি হয়, আর CSS দিয়ে সেটাকে সুন্দর করা হয়।

<img> ট্যাগ ব্যাখ্যা করো। অনুধাবন

HTML পেজে <img> ট্যাগ দ্বারা ছবি নির্ধারণ করা হয়। এটি শূন্য ট্যাগ — অর্থাৎ শুধু Attribute বহন করে, কোনো Closing tag নেই। পেজে ছবি ব্যবহার করতে হলে src (source) Attribute ব্যবহার করতে হবে। ছবি যুক্ত করলে পেজ আকর্ষণীয় হয়।
